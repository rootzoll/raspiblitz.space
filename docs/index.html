Your Lightning & Bitcoin full node on a Raspberry Pi

Your financial sovereignty starts here

RaspiBlitz is a do-it-yourself Lightning node running together with a Bitcoin full node on a Raspberry Pi and a nice display for easy set up & monitoring.

Get the latest version

Features

  • Sovereignty

    RaspiBlitz will do the heavy lifting for you to set up your Bitcoin full node and Lightning node. You will own your data.

  • Low cost

    RaspiBlitz is a first-class Raspberry Pi citizen. That offers you to run a node with minimal investment in the setup and maintenance phase.

  • Monitoring

    With the possibility to run RaspiBlitz with a display, you have all the important node metrics in sight.

  • WebUI

    With our new WebUI, it's easier than ever to set up and maintain your node. If you love the terminal, we covered you as well.

  • Broad Integration

    RaspiBlitz supports many apps and services which are easy to enable.

  • Community

    RaspiBlitz is an open-source project powered by hundreds of contributors making thousands of individual contributions.

Supported apps

RaspiBlitz plays nice. It supports many different apps and services. From Tor to Sphinx Relay. It's easy to switch your favorite apps on.
